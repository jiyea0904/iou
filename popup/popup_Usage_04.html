How to use 'Data Analysis' of IP firms

1. Why is 'Data Analysis' of IP Firms necessary?
  • Because this analysis helps you to find the best foreign partner in the easiest and most precise way.
  • Because this analysis helps you to check the credibility of the foreign firms you are interested in.
2. What kinds of Data Analysis IOU uses?
  • IOU offers you three unique methods on the ‘DATA-ANALYSIS’ corner on IOU Website.
  • 1) Precise Analysis
    • Analyze dealing history precisely by country. (whole lists are provided)
    • Analyze the data by Incomings, Outgoings, and Ins + Outs.
  • 2) Integrated Analysis
    • Shows the integrated data of whole countries.
    • This is called Balancing-Table.
  • 3) Navigating Analysis
    • Navigate the IP firms having a high possibility of exchange.
    • The users can easily find good partners, using IOU's Navigation Tool.
  • It is very useful to analyze data by ① country, ② partner firm, ③ number of incomings, ④ number of outgoings, and ⑤ the exchange rate.
    If the exchange rate is 100%, it means the ideal '1:1 IP-Exchange' is made.
  • Definition of exchange rate = (Incoming/Outgoing) X 100 (%) (If the denominator is 0, replace it with 1.)
